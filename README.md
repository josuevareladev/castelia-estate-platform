# Castelia Estate Platform 🌿
**Premium Real Estate Management System by Castelia Studio**

Castelia Estate Platform is a high-end MERN stack application designed for modern real estate management. It combines a sophisticated aesthetic with robust backend logic, providing a seamless experience for administrators to manage properties and digital assets in the cloud.

## 💎 Premium Features
- **Full CRUD Management:** Comprehensive control over property listings.
- **Secure Authentication:** Protected administrative routes using JSON Web Tokens (JWT).
- **Cloud Asset Hosting:** Real-time image processing via Cloudinary API.
- **Advanced UI/UX:**
    - **Skeleton Loaders:** Professional loading states.
    - **Real-time Filtering:** Instant search by title or location.
    - **Pagination:** Optimized catalog loading.
- **Responsive Design:** Fully optimized for desktop and mobile.

## 🛠 Tech Stack
- **Frontend:** React.js, Tailwind CSS, Axios.
- **Backend:** Node.js, Express.js.
- **Database:** MongoDB Atlas.
- **Cloud Services:** Cloudinary.
